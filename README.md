# Проектная работа модуля 6. Рефакторинг кода на Rust

## Сборка проекта

```
cargo build
```

## Запуск проекта

```
cargo run example.log
```

## Тестирование проекта

Запуск тестов:

```
cargo test
```

## Генерация HTML-отчета о покрытии:

```
cargo llvm-cov --html
```

## Документация

Сгенерировать документацию:

```
cargo doc
```
